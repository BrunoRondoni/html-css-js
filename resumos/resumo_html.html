Usando listas em HTML

Lista de dados não ordenada <ul>

Use a tag <ul> junto com a tag <li> para criar listas não ordenadas.

<ul>
  • Nome LI <li>
  • RG <li>
  • Cpf <li>
  • Endereço <li>
<ul>

Lista de dados ordenada <ol>

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

<ol>
  1. Nome LI <li>
  2. RG <li>
  3. Cpf <li>
  4. Endereço <li>

Tags de ênfase em HTML

<strong> strong Usada para definir o texto de grande importância. O conteúdo interno normalmente é exibido em negrito .

<b> bold Especifica texto em negrito sem transmitir qualquer importância extra ou relevância. Ele é geralmente usado para destacar palavras-chaves em um resumo

<em> emphasis usada para definir o texto enfatizado. O conteúdo interno é normalmente exibido em itálico. Um leitor de tela pronunciará as palavras com ênfase, usando ênfase verbal.

<i> itálico Define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno é normalmente exibido em itálico .freqüentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.

Divisores de conteúdo em HTML

<div> É um container genérico para conteúdo de fluxo, que de certa forma não representa nada. Ele pode ser utilizado para agrupar elementos para fins de estilos (usando class ou id), ou porque eles compartilham valores de atributos, como lang. Ele deve ser utilizado somente quando não tiver outro elemento de semântica (tal como <article> ou <nav>).

<section> Representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.

<article> Representa uma composição independente em um documento, página, aplicação, ou site, ou que é destinado a ser distribuido de forma independente ou reutilizável, por exemplo, em sindicação. Este poderia ser o post de um fórum, um artigo de revista ou jornal, um post de um blog, um comentário enviado por um usuário, um gadget ou widget interativos, ou qualquer outra forma de conteúdo independente.
